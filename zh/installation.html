如何安装¶

依赖组件

  1. Python 3.6 or 3.7

  2. CUDA >= 10.0

  3. cuDNN >= 7.6

  4. APEX

  5. PyTorch >=1.2

  6. 对于多GPU或者分布式训练，推荐安装： NCCL >= 2.4

Tip

您还可以不安装这些依赖而直接使用 英伟达的PyTorch镜像 . 在该镜像中，所有的依赖都已提前为您安装好。方便您直接使用。

安装NeMo及其Collections

  1. 克隆代码库：

git clone https://github.com/NVIDIA/nemo

  1. 切换到 nemo 文件夹并运行： python setup.py install

  2. 安装collections

    1. ASR: 进入 collections/nemo_asr 文件夹并运行 sudo apt-get install libsndfile1 && python setup.py install

    2. NLP: 进入 collections/nemo_nlp 文件夹并运行 python setup.py install

    3. LPR: 进入 collections/nemo_simple_gan 文件夹并运行 python setup.py install

对于开发模型，请运行 python setup.py develop

  1. 运行单元测试以验证是否安装成功：

python -m unittest tests/*.py

  1. 切换到 examples/start_here 文件夹运行一些范例。
